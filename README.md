# RDSMNet  
Enhancing Photogrammetric DSM Based on Multiscale and Domain-Invariant Semantic Feature Learning

This repository provides the **official implementation** of **RDSMNet**, a deep learning framework for refining photogrammetric Digital Surface Models (DSMs) using multiscale semantic features and domain-invariant learning strategies.
